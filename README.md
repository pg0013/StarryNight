## 起動方法  
gameフォルダ内のexeファイルをダブルクリックでゲームが起動します。  
## 終了方法
タイトル画面で終わりを選択する  
## 操作説明
* Lスティック　　移動  
* Rスティック　　カメラ移動  
* Aボタン　　　　ジャンプ  
* Rトリガー　　　射撃モード（星を連れている状態でRトリガーを押し込むと、射撃モードに移行します。右スティックで射撃標準を移動することができます）  
* Yボタン　　　　星を選択（射撃モードの状態でＹボタンを押すと、星をパチンコから発射することができます）  
* Bボタン　　　　タイミングを決定（タイミングゲームの状態でBボタンを押すと、収縮する円を停止することができます）  
* STARTボタン　　ポーズメニューの表示  
## 概要説明
ステージにある星を集めて、発射ポイントから集めた星を星座に撃ち、星座を完成させます。ステージにある星を一度にすべて集めると高得点になります。  
ただし、敵に当たると獲得した星は散らばるので注意してください。
